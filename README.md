## 🧩 Sobre o Projeto 👩‍💻

Este projeto consiste em um sistema bancário simples desenvolvido em **Python**, executado diretamente pelo terminal.

O sistema permite realizar operações básicas como depósitos, saques, consulta de extrato, cadastro de usuários e criação de contas bancárias.

O projeto foi desenvolvido com o objetivo de praticar conceitos fundamentais da linguagem Python, como funções, listas, dicionários, organização de código e passagem de parâmetros.

## ⚙️ Funcionalidades

**💰 Operações Bancárias**

- Realizar depósitos
- Realizar saques
- Consultar extrato
- Exibir saldo atualizado

**🏧 Sistema de Saques**

- Limite de R$ 500,00 por saque
- Máximo de 3 saques por execução
- Impede saques sem saldo suficiente
- Validação de valores inválidos

**👤 Usuários**

- Cadastro de novos usuários
- Verificação de CPF duplicado
- Armazenamento de nome, CPF, data de nascimento e endereço

**🏦 Contas Bancárias**

- Criação de contas vinculadas a usuários
- Agência fixa (0001)
- Numeração automática das contas
- Listagem de todas as contas cadastradas

## 🖥️ Como Executar

**1** - Certifique-se de possuir o **Python 3** instalado.

**2** - Clone este repositório:

```bash
git clone https://github.com/Caetano-2012/NOME-DO-REPOSITORIO.git
```

**3** - Entre na pasta do projeto:

```bash
cd NOME-DO-REPOSITORIO
```

**4** - Execute o programa:

```bash
python main.py
```

**5** - Utilize o menu do sistema:

- **d** → Depositar
- **s** → Sacar
- **e** → Exibir extrato
- **nu** → Novo usuário
- **nc** → Nova conta
- **lc** → Listar contas
- **q** → Sair
